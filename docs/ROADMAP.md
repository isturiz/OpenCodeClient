# Roadmap

## Milestone 1 — Native vertical slice

- Multi-server settings with Keychain credentials
- Projects and sessions grouped by OpenCode directory
- REST and SSE chat with tool, reasoning, status, and permission states
- Dynamic model and agent selection
- FluidVoice capture, transcription, and optional post-processing
- Adaptive iPhone and iPad interface, localization, accessibility, tests, and CI

## Milestone 2 — Review workflows

- File and directory browsing
- File cards, content preview, Markdown-relative images, and session diffs
- Attachments, session fork/revert, richer command support, and searchable history
- More complete tool-specific rendering based on captured OpenCode fixtures

## Milestone 3 — Remote operations

- Bonjour discovery and optional SSH transport
- Push notifications or Live Activities for blocked sessions
- Share links, deeper session management, and App Store distribution automation
- Evaluate a dedicated macOS relay only if secure tunnels are insufficient for community users

The roadmap follows released OpenCode contracts. Features are not implemented against undocumented or
removed endpoints merely because an older client used them.
